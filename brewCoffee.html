/Pseudocode/

What is pseudocode?

Pseudocode is used as programming technque to describe in English term the tasks that the program will excute. Writing pseudocode saves time during the development as well during the testing of the code. Usually a pseudocode is written in if statatements. So in this scenario we will write a simple and basic pseudocode for our coffee machine.

  1. Check if the coffee machine has cofee coffee beans
    if has coffee beans and the coffee beans is the right amount
    return true
    else reuturn false
  2. Check if the level of water is right
    if the level of water is "x" and the temperature is "c" return true
    else if the water level is below "x" return false and message
    else if the water level is above "x" return false and message
  3. Check if the water temperature is 96 degrees celsius
    if the water temperature is 96 deg return true
    else if the water temperature is above 96 degress return false and message, return to the beginning of the statement.
    if water temperature if below 96 deg return false, message, and heat to the right temperature.
  4. check if the coffee ig grindede
    if the coffee if grinded return true
    else return false

The above example is just a simple and brief example about pseudocode. As you can see the logic is based on true and false statements

{Javascript code}

Now comes the fun part. Lest's write the Javascript code to run the machine's logic. Here we will keep it very simple and basic

First we declare two variables. Variables in programming are names used to hold values. In the example below the first variable called coffeeGrain holds a value of 10, which means that is the rigth amout of coffee grain for the machine

The second variable is an object. Object in Javascript contains properties like a car, where has color, weight and so on. The variable called glassOfWater, contains two properties: quantity, measures the amout of water, in this case the value is 200, and temperature, which is 96deg

Javascript

            
let coffeeGrain = 10
const glassOfWater = {
quantity: 200,
temperature: 99
}

Below we have a function. Functions are a set of statements that performs a task. The task that the function below performs is to check if the variable called coffeeGrain contains the rigth amount, then returns true. In other words, if the amout of coffeeGrain is 10, return true to turn on the grinder.

Inside the function, switch statement is been used, in which if the first case is false the program will look for the next until reaches the last option. The default option runs if any of other cases don't matches the condition.

Javascript

            
function turnOnTheGrinder(coffeeGrain) {
  switch (true) {
    case (coffeeGrain === 10):
      return true
      break;
    case (coffeeGrain < 10):
      console.log("add more grains")
      return false
      break;
    case (coffeeGrain > 10):
      console.log("please remove grain")
      return false
      break;
    default:
      return false
  }
  
}

The next block of code we have a function that is going to check the water and temperature. Using the variables that were assingned in the beginning, checks if the water level is the right one using comparison operators. Comparison operators, compares two values and return true if the comparison match. As the first example below, if the quantity of water (water.quantity) is equal 200, by the equality operator ===, "and" using the operator && if the water temperature (water.temperature) is equal 96 return true.

As you can see we check all the possible conditions to avoid any crack in our software.
